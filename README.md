## Estacionamento - API

## Instalando as dependencias

```bash
$ npm install
```
## ENV

Crie o arquivo .env na raiz do projeto, copie e coloque os atributos do arquivo .envsample

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
